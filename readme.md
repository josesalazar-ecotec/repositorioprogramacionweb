# Mi Página Web Personal

Este proyecto consiste en el desarrollo de una página web básica utilizando HTML y CSS como parte del aprendizaje de programación web.

## Desarrollo del proyecto

Como estudiante, desarrollé esta página web paso a paso.

Primero creé la estructura base en HTML utilizando etiquetas semánticas como `header`, `main`, `section` y `footer`.

Después agregué las secciones principales de la página: Sobre mí, Intereses, Formación y Contacto. También incorporé un menú con enlaces internos para poder navegar dentro de la misma página.

Luego añadí una imagen personal y un formulario de contacto con campos de nombre, correo y mensaje.

Finalmente, apliqué estilos básicos con CSS para mejorar la presentación de la página, utilizando color de fondo, color de texto y márgenes.

## Tecnologías utilizadas

- HTML
- CSS
